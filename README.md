# Unofficial-Monoblocks-Rewritten
Monoblocks mod originally made by goldensilver853. This is a minecraft block mod that was rewritten to be able to maintain and add support for different block shapes. This was decompiled and had code rewritten for easier maitaining. 

This is a fan-made rework of the mod to improve compatibility and usability while also adding vanilla slabs, stairs, and more. 
For ease of maintenance, this remake requires that the Foxblocks be present.
